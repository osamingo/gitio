# gitio

gitio comannd generates short url by [git.io](https://git.io)

## Installation

```
$ go get -u github.com/osamingo/gitio/cmd
```

## Usage

```
$ gitio -code=gitio-cmd https://github.com/osamingo/gitio
https://git.io/gitio-cmd
```
